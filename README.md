This repository contains code for the Applied Soft Computing Journal paper:

Chao Zhou, Xiaowen Shi. Yuan-Gen Wang. Query-Efficient Hard-Label Black-Box Attack against Vision Transformers.

To run simp:
```
python run_simp.py --data_root <imagenet_root> --num_runs 100 --max_iters 4000  --patch_size 16 --model DeiT_T
```
